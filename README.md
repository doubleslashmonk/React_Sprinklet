# React-Sprinket

Sprinklet Version 0.0.1

This app fetches data using Github APIs and renders it using React. The notes component uses Firebase to store and dispaly data in real time.

### Tools and technologies used
NPM
WebPack
Firebase
Facebook React


### Instructions

```bash
npm install
webpack -w
npm start
```
